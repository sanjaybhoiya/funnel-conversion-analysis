# 🛒 User Funnel & Conversion Analysis

## Overview
End-to-end analysis of the Google Merchandise Store user journey 
using BigQuery SQL, Python, and Power BI. Identified key conversion 
bottlenecks across 250,000+ user events and delivered actionable 
business recommendations.

## 📊 Key Findings
- Overall conversion rate: **1.6%** (page view → purchase)
- **77% drop-off** at top of funnel — users land but don't view products
- Only **20%** of product viewers add to cart
- **35%** of cart users complete purchase — abandoned cart opportunity
